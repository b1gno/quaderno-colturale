# 🚀 AVVIO SILENZIOSO - Quaderno di Campo

## 📥 File che hai ricevuto:

1. **Avvia (Silenzioso).vbs** - Avvio con messaggio finale ✉️
2. **Avvia (Invisibile).vbs** - Avvio 100% silenzioso 👻
3. **Chiudi Quaderno.vbs** - Ferma tutto con un clic 🛑

---

## ✨ QUALE USARE?

### 🔔 Avvia (Silenzioso).vbs - CONSIGLIATO
- Avvia tutto in background
- Nessun terminale visibile
- Messaggio finale: "Avvio completato"
- **Ideale per uso normale**

### 👻 Avvia (Invisibile).vbs - ULTRA SILENZIOSO
- Avvia tutto in background
- Nessun terminale
- NESSUN messaggio
- Si apre solo il browser
- **Ideale per presentazioni/demo**

### 🛑 Chiudi Quaderno.vbs
- Ferma Node.js
- Ferma MySQL
- Ferma Apache
- Messaggio di conferma
- **Usa questo per chiudere tutto**

---

## 🎯 COME USARE

### Setup iniziale (una volta sola):

1. **Scarica** i 3 file VBS
2. **Metti** nella cartella principale del progetto:
   ```
   quaderno-campo/
   ├── Avvia (Silenzioso).vbs
   ├── Avvia (Invisibile).vbs
   ├── Chiudi Quaderno.vbs
   ├── backend/
   └── frontend/
   ```

### Per avviare:
- **Doppio clic** su `Avvia (Silenzioso).vbs` OPPURE `Avvia (Invisibile).vbs`
- Aspetta 10 secondi
- Si apre automaticamente il browser
- **Fatto!** ✅

### Per chiudere:
- **Doppio clic** su `Chiudi Quaderno.vbs`
- Conferma con "Sì"
- **Tutto chiuso!** 🛑

## 🎨 AGGIUNGI ICONE PERSONALIZZATE

### Passo 1: Scarica un'icona
Vai su [icons8.com](https://icons8.com) e cerca:
- "agriculture" 🌾
- "farm" 🚜
- "field" 📊
- "notebook" 📝

Scarica in formato **PNG** (qualsiasi dimensione, meglio 256x256)

### Passo 2: Converti in .ico
- Vai su [convertio.co/png-ico](https://convertio.co/png-ico)
- Carica la tua immagine PNG
- Scarica il file .ico
- Salvalo come `icona.ico` nella cartella del progetto

### Passo 3: Crea collegamento con icona

#### Per "Avvia (Silenzioso).vbs":
1. Tasto destro su `Avvia (Silenzioso).vbs`
2. **Invia a** → **Desktop (crea collegamento)**
3. Vai sul Desktop
4. Tasto destro sul collegamento → **Proprietà**
5. Clicca **Cambia icona**
6. Clicca **Sfoglia** → Seleziona `icona.ico`
7. **OK** → **Applica**
8. Rinomina in: **🌾 Quaderno di Campo**

#### Per "Chiudi Quaderno.vbs":
Stessa procedura, ma rinomina in: **🛑 Chiudi Quaderno**

## 🔧 RISOLUZIONE PROBLEMI

### ❌ Errore: "XAMPP non trovato"
**Soluzione:** Modifica lo script VBS e cambia il percorso:
```vbs
xamppPath = "C:\xampp"  ' ← Cambia con il tuo percorso
```

### ❌ Il browser non si apre
**Soluzione:** 
1. Aspetta 15 secondi dopo il doppio clic
2. Apri manualmente: `http://localhost:3000`

### ❌ "Server già in esecuzione"
**Soluzione:** 
1. Usa `Chiudi Quaderno.vbs` prima
2. Poi riavvia con `Avvia (Silenzioso).vbs`

### ❌ L'icona non si vede
**Soluzione:**
1. Riavvia Esplora Risorse (CTRL+SHIFT+ESC → Esplora Risorse → Riavvia)
2. Oppure riavvia il PC

## 🔄 AVVIO AUTOMATICO CON WINDOWS

Se vuoi che si avvii all'accensione del PC:

1. Premi **Win + R**
2. Digita: `shell:startup`
3. Premi **Invio**
4. Copia il collegamento di `Avvia (Invisibile).vbs` in questa cartella
5. **Fatto!** Si avvia automaticamente all'accensione

**ATTENZIONE:** Usa `Avvia (Invisibile).vbs` per l'avvio automatico, così non vedi messaggi fastidiosi.
